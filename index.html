▶ How to run the scripts on your machine:

    
1. Install dependencies (once):  pip install pywin32 pandas

    
2. Run AAS extraction:  python extract_aas.py

    
3. Run Fabric extraction:  python extract_fabric.py

    
4. Each script outputs a .csv file in the same folder — upload them below to compare.

    
⚠ Requires Windows with MSOLAP provider installed (comes with Power BI Desktop or SSMS).
